# Testing — verify-before-ship

Загружается всегда (нет `paths`). Стек-независимо.

- **Sensor (PostToolUse) запускает тесты сам** при правке в `WATCH_DIR`. Mute the green:
  при успехе молчит, при провале — выводит ошибку. Не дублируй ручным прогоном без нужды.
- **Тест = критерий готовности.** Задача не закрыта, пока проверка не зелёная.
- **Осознанно меняешь поведение → обнови ожидания теста/снапшота**, не подгоняй тест
  под баг.
- **Не можешь проверить автоматически → проверь руками** (запуск, скриншот) и скажи об этом
  явно. Не выдавай непроверенное за готовое.
