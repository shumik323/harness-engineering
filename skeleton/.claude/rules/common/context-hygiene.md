# Context Hygiene — гигиена контекстного окна

Загружается всегда (нет `paths`). Стек-независимо. CORE-слой.

- **Источник истины — файлы, не история чата.** План, задачи, состояние держи в файлах
  (план/overview/доки); читай их, не полагайся на прокрутку истории. История сжимается — файл нет.
- **`/compact` проактивно, на границе задачи.** Длинная сессия вытесняет правила и контекст
  (context rot). Сжимай до того, как поплыло, не когда уже потерял нить.
- **JIT-загрузка, не всё впрок.** Тяжёлые доки читать по требованию (Read), не тащить в старт-контекст.
  `@import` НЕ лениво — грузит на старте, для JIT не годится. Always-loaded держи тонким (см. корневой CLAUDE.md).
- **Подсвечивай деградацию, не действуй по памяти.** Подозрение, что правило/факт выпали из окна —
  скажи и перечитай с диска.
