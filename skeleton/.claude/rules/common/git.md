# Git — этикет коммитов

Загружается всегда (нет `paths`). Стек-независимо.

- **Коммит только по явному запросу** пользователя («commit»). Не коммитить самовольно.
- **Не коммитить в дефолтную ветку напрямую** — сначала ветка.
- **Readonly-зоны** (`READONLY_ZONES` из `.harness.conf`) — только через build-команды.
  Guard блокирует прямую запись (exit 2). См. `block-zones.sh`.
- **Не менять `package.json` / манифест зависимостей** без явного запроса.
- `gh` CLI для GitHub-операций (PR, issues).
