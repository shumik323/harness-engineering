# Go — правила (грузятся только на *.go)

> STUB-пример языкового слоя. Показывает паттерн «common + per-language».
> Заполни под свой Go-проект.

- **Расширяет `common/*`.** Здесь только Go-специфика, общее не дублировать.
- `gofmt` / `goimports` — формат не обсуждается.
- Ошибки — явная проверка `if err != nil`, не глотать.
- Тесты рядом: `xxx_test.go`, table-driven где уместно.
- Экспортируемое — с doc-комментарием.

> Реестр найденных ловушек проекта — `.claude/docs/gotchas.md`.
