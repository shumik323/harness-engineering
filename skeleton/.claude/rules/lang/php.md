# PHP — правила (грузятся только на *.php)

> STUB-пример языкового слоя. Показывает паттерн «common + per-language».
> Заполни под свой PHP-проект.

- **Расширяет `common/*`.** Здесь только PHP-специфика, общее не дублировать.
- `declare(strict_types=1)` в начале файла.
- PSR-12 формат, `php-cs-fixer` / `phpcs`.
- Типизация аргументов и возврата везде, где возможно.
- Тесты — PHPUnit, рядом или в `tests/`.

> Реестр найденных ловушек проекта — `.claude/docs/gotchas.md`.
