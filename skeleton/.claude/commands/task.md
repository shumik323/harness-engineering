# task

Шаблон промпта для новой задачи агенту. Заполни секции перед отправкой.

## Использование

```
/task
```

Скопируй шаблон ниже, заполни все секции, удали комментарии.

---

## Шаблон

```markdown
## Контекст (что уже есть)

<!-- Стек, архитектура, что важно знать до выполнения задачи.
     Что агент должен учесть, чтобы не сломать существующее. -->

## Задача

<!-- Что конкретно создать или изменить. Одним абзацем. -->

## Детали реализации

<!-- Поля, типы, паттерны которые нужно соблюдать.
     Ссылки на эталоны: @путь/к/файлу -->

## Ожидаемый результат

<!-- Что должно работать после выполнения.
     Конкретные критерии: "POST /X возвращает Y", "тест Z зелёный" -->

## Ограничения

<!-- Что НЕ нужно делать.
     Что нельзя трогать. Какие зависимости не добавлять. -->
```

## Пример (заполненный)

```markdown
## Контекст

Vue 3.5 component library, стек: TS 5.7 + Vite 6 + Tailwind 3 + Vitest.
Эталон структуры: `packages/ui-kit/lib/components/md-dropdown/`.
Guard блокирует запись в `dist/` и `storybook-static/`.

## Задача

Создать компонент MdChip — тег/метка с текстом и опциональной кнопкой закрытия.

## Детали реализации

Props: `label: string`, `closable?: boolean`.
Emits: `close`.
Классы: `use-chip-classes.ts` → `computed<string[]>`.
Маркер QA: `md-chip` на корневом элементе.

## Ожидаемый результат

- Компонент рендерит текст из `label`
- При `closable=true` показывает кнопку ×, клик → emit `close`
- Тест snapshot зелёный
- Story создана

## Ограничения

Не использовать inline styles — только Tailwind.
Не трогать `md-dropdown` как эталон — только читать.
Не добавлять зависимости.
```

---

## Почему это работает

Четыре компонента (из курса Claude Code):
- **Контекст** — агент понимает систему, не делает глобальные категории вместо привязанных к пользователю
- **Задача** — конкретная цель, не "улучши компонент"
- **Детали** — паттерны зафиксированы, агент не изобретает
- **Ограничения** — агент не переписывает рабочий код и не тащит лишнее

Без ограничений агент может сделать правильно, но попутно сломать что-то рядом.
